# c++面试宝典
#### 面向对象本质是设计并扩展自己的数据类型。设计自己的数据类型就是让类型与数据匹配。
#### 内置的c++类型分两组：基本类型和复合类型。基本类型为整数和浮点数。复合类型为数组，字符串，指针和结构。
